Dubai-real-estate-analysis

📊 Overview

This project focuses on analyzing real estate data in Dubai to understand pricing trends across different areas and property types. The goal is to derive meaningful insights that can help buyers, investors, and businesses.

🎯 Objectives

* Compare average property prices across different areas
* Analyze price per square foot
* Understand the impact of size and location on property prices
* Identify affordable vs premium locations

🛠 Tools Used

* Microsoft Excel
* Pivot Tables
* Charts (Bar, Pie, Scatter)
* Slicers (for interactivity)

📁 Dataset

The dataset includes:

* Area
* Price (AED)
* Bedrooms
* Size (sq ft)
* Property Type (Apartment/Villa)

📈 Key Insights

* Palm Jumeirah has the highest average property prices and price per sq ft
* Downtown Dubai is also a premium location with high pricing
* Jumeirah Village Circle offers more affordable housing options
* Property prices increase with size, but location plays a major role
* Villas are slightly more in number in this dataset and are generally priced higher

📊 Dashboard Features

* Area-wise price comparison
* Price per square foot analysis
* Price vs Size scatter plot
* Property type distribution
* Interactive filters using slicers

🚀 Conclusion

This project demonstrates fundamental data analysis skills including data cleaning, transformation, and visualization using Excel. It highlights how data can be used to understand real estate market trends.
